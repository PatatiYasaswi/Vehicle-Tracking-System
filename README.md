# Vehicle-Tracking-System
The Vehicle Tracking and Logging System is a web-based application designed to automate vehicle entry and exit monitoring using camera-based detection. The system captures vehicle images, detects number plates using Optical Character Recognition (OCR), and stores vehicle details in a database.
- It provides a user-friendly dashboard for monitoring vehicle activity, managing records, and generating reports. The system improves accuracy, reduces manual effort, and enhances security in areas such as campuses, apartments, and offices.

## Features
- Automatic vehicle image capture
- Number plate detection using OCR
- Vehicle color detection
- Web-based dashboard
- Manual vehicle entry option
- Vehicle exit tracking
- Vehicle records management
- Export data to Excel
- Live camera monitoring

## How It Works
1)Vehicle image is captured using camera
2)Image is processed using OpenCV
3)Number plate is detected using OCR
4)Vehicle color is identified
5)Data is stored in MySQL database
6)Information is displayed on dashboard
7)User can search, manage, and export records
